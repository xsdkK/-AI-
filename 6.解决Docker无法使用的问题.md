***引言：当我们在使用Docker去Docker Hub拉取镜像时，经常会发现特别慢会卡住不动的情况，此时极大可能不是Docker的问题，而是网络环境的问题***

**验证Docker的可用性**

    systemctl status docker
    发现结果返回active(running),则说明Docker服务本身是没有问题的

**修改镜像源**

    打开docker下的daemon.json文件，进去极有可能会看到registry-mirrors镜像源是官方源，此时我们要将其修改为国内可用源，
    比如，https://docker.1ms.run或者https://docker.xuanyuan.me
    国内镜像源有很多，可自行查找替换

**修改DNS配置**

    echo -e "nameserver 223.5.5.5\nnameserver 223.6.6.6\nnameserver 114.114.114.114" > /etc/resolv.conf

**测试**

    运行 docker run hello-world
    此时Docker会从Docker Hub里拉取hello-world运行
